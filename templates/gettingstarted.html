Using the pattern library

The pattern library comes with one Javascript file and three CSS files (old IE, IE9 and main - these'll be explained in more detail).

Download or link to the files from:

  • https://onsdigital.github.io/sixteens/css/old-ie.css
  • https://onsdigital.github.io/sixteens/css/ie-9.css
  • https://onsdigital.github.io/sixteens/css/main.css
  • https://onsdigital.github.io/sixteens/js/main.js

These files should be wrapped in conditional statements. So, in the end it should look like this:

                    <!--[if lte IE 8]><link rel="stylesheet" href="https://onsdigital.github.io/sixteens/css/old-ie.css"><![endif]-->
                        

                        <!--[if IE 9]><link rel="stylesheet" type="text/css" href="https://onsdigital.github.io/sixteens/css/ie-9.css"><![endif]-->

CSS

Each of the three CSS file should be included in the head of your page. There are three CSS files because:
  • old-ie.css removes media queries so that styling doesn't break in IE8.
  • ie9.css is split into two files to get around the IE9 CSS selector limit.
  • main.css is the ordinary CSS which will be applied to all modern browsers.

Javascript

Add a link (or download to your own project) the main CSS file - https://onsdigital.github.io/ons-pattern-library-starter/css/main.css) in the head of your page. Put a link to the pattern library JS file from ons.gov.uk ()

How is the pattern library built?

The patterns are built in the 'Sixteens' repository on the ONSDigital Github. To have a play around with the original code follow the instructions in the README to install and use the pattern library.
